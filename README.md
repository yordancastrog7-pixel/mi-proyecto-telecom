##  Descripción del Proyecto
Este repositorio contiene la evolución técnica y analítica del sistema de facturación para el **Proyecto Telecom**. A lo largo de 15 semanas, se ha estructurado una base de datos relacional robusta, evolucionando desde la creación de esquemas fundamentales hasta la implementación de consultas avanzadas para el análisis de tendencias, jerarquías y métricas temporales.

El proyecto demuestra el dominio de **MySQL** y técnicas de optimización de consultas, incluyendo el uso de estructuras recursivas, funciones de ventana (window functions) y encapsulamiento mediante vistas.

##  Instrucciones de Despliegue
Para replicar este entorno de trabajo:
1. Asegúrese de tener **Docker** instalado en su máquina.
2. Clone este repositorio: `git clone <https://github.com/yordancastrog7-pixel/mi-proyecto-telecom>`
3. Despliegue el contenedor con: `docker-compose up -d`
4. Ejecute los archivos `.sql` correspondientes a cada semana desde su gestor de base de datos preferido.
